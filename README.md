# My Survival Shooter

Chapter 3 of "Game Programming in C++" modified into a survival shooter game.
(「ゲームプログラミングC++」の第3章をベースに改造したサバイバルシューティングゲームです)

## Features (実装した機能)
- **Infinite Enemy Spawning**: Enemies spawn randomly from outside the screen.
  (画面外からのランダムな敵スポーン)
- **Player Movement**: Rotation using A/D keys.
  (A/Dキーによる旋回操作)
- **Shooting**: Fire rockets with SPACE key.
  (スペースキーによる発射)
- **Collision Detection**: Circle-based collision logic.
  (円形の当たり判定)

## Controls (操作方法)
- **Rotate**: A / D
- **Shoot**: Space
- **Quit**: Esc
